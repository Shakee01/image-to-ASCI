Image to ASCII Converter

A simple Python project that converts images into ASCII art.

Description

This project converts an image into ASCII art by replacing each pixel with a character based on its brightness.

The main objective was to practice basic image processing, working with loops, and transforming data into a text-based format.
The output is a visual representation of the image made entirely of ASCII characters.

Why I Built This

The project was created to improve Python programming skills, particularly:

working with external libraries (Pillow)

understanding how images are represented as pixels

using loops and conditions

converting data into a different format

Features

Convert images into ASCII art

Adjustable output width

Grayscale image conversion

Save result to a text file

Display ASCII art in terminal

Technologies

Python 3

Pillow (PIL)

How to Run

Install required library:

pip install pillow

Place your image in the project folder (e.g. image.png)

Run the script:

python main.py
